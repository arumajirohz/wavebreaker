# Wavebreaker

Wavebreaker is a fast-paced elemental tower defense game built using HTML5 Canvas.

## Features

- 🔥 Fire, ❄️ Ice, ☣️ Poison towers
- Dynamic wave progression
- Boss enemies (👾 🐍 💀)
- Combo-based rewards system
- Procedural path generation
- Lightweight single-file architecture

## Gameplay

Place towers strategically to control enemy flow and survive increasingly difficult waves.  
Each run presents a new layout and new challenges.

## Tech

- Vanilla JavaScript
- HTML5 Canvas
- No external dependencies

## Controls

- Tap / Click to place towers
- Tap towers to upgrade
